## 监督学习 vs 强化学习

| 维度 | 监督学习 | 强化学习 |
|------|----------|----------|
| 数据 | 有标签 (x, y) | 无标签，只有交互轨迹 |
| 反馈 | **即时**反馈（每个样本有标准答案） | **延迟**反馈（奖励/惩罚可能滞后） |
| 目标 | 最小化预测误差 | 最大化长期累积奖励 |
| 决策 | 单步、独立样本 | 序列决策，前后状态相关 |
| 典型任务 | 分类、回归 | 控制、博弈、导航 |

## 马尔可夫决策过程（MDP）

强化学习的数学框架：**MDP = (S, A, P, R, γ)**

- **S**：状态空间
- **A**：动作空间
- **P(s'|s,a)**：状态转移概率
- **R(s,a)** 或 **R(s,a,s')**：奖励函数
- **γ ∈ [0,1]**：折扣因子，平衡「眼前奖励」与「未来奖励」

#### 随机过程与随机现象
系统在时间上演化，下一状态具有随机性；我们关心的是**概率分布**而非单条轨迹。

#### 马尔可夫性质
\[
P(S_{t+1} \mid S_t, S_{t-1}, \ldots) = P(S_{t+1} \mid S_t)
\]
**当前状态已包含做决策所需的全部历史信息**——未来只依赖现在，不依赖更早的过去。

#### 马尔可夫过程（MP）与马尔可夫链
满足马尔可夫性质的随机过程；状态空间有限且离散时，称为**马尔可夫链**。

#### 状态转移矩阵
对离散状态，用矩阵 **P** 描述一步转移：
\[
P_{ij} = P(S_{t+1}=j \mid S_t=i), \quad \sum_j P_{ij} = 1
\]
矩阵乘法可计算多步转移概率。

#### 马尔可夫奖励过程（MRP）与回报
在 MP 上加入奖励 \(R_{t+1}\)，定义**回报**（Return）：
\[
G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots
\]
γ 越小越「短视」，γ 越接近 1 越重视长期。

#### 价值函数
- **状态价值** \(V(s) = \mathbb{E}[G_t \mid S_t=s]\)：从状态 s 出发，按某策略 π 能期望获得多少总奖励
- **动作价值** \(Q(s,a) = \mathbb{E}[G_t \mid S_t=s, A_t=a]\)：在 s 下先执行 a，再按 π 继续

#### MRP 闭式解
对有限状态 MRP，令 **r** 为期望奖励向量，**P** 为转移矩阵：
\[
\mathbf{V} = (\mathbf{I} - \gamma \mathbf{P})^{-1} \mathbf{r}
\]
（当 \((\mathbf{I}-\gamma\mathbf{P})\) 可逆时成立）

#### Bellman 期望方程
价值函数满足递归关系——**当前价值 = 即时奖励 + 折扣后的下一状态期望价值**：
\[
V^\pi(s) = \sum_a \pi(a|s) \sum_{s',r} p(s',r|s,a)\big[r + \gamma V^\pi(s')\big]
\]
\[
Q^\pi(s,a) = \sum_{s',r} p(s',r|s,a)\big[r + \gamma \sum_{a'} \pi(a'|s') Q^\pi(s',a')\big]
\]

#### Bellman 最优方程
最优策略 π* 对应的最优价值：
\[
V^*(s) = \max_a \sum_{s',r} p(s',r|s,a)\big[r + \gamma V^*(s')\big]
\]
\[
Q^*(s,a) = \sum_{s',r} p(s',r|s,a)\big[r + \gamma \max_{a'} Q^*(s',a')\big]
\]
最优策略：\(\pi^*(s) = \arg\max_a Q^*(s,a)\)

#### POMDP 建模
**部分可观测 MDP**：智能体看不到真实状态 S，只能得到观测 O。
- 引入**信念状态**（belief state）\(b(s) = P(S=s \mid o_{1:t}, a_{1:t-1})\)
- 决策问题仍可在扩展状态空间上转化为（近似）MDP / 动态规划问题

## 策略优化与动态规划

给定 MDP，求最优策略 π* 的经典思路：**策略评估 → 策略改进 → 迭代至收敛**。

#### 策略优化（Policy Improvement）
若 \(Q^\pi(s, \pi'(s)) \ge V^\pi(s)\) 对所有 s 成立，则 π' 不差于 π。
**贪心改进**：\(\pi'(s) = \arg\max_a Q^\pi(s,a)\)

#### 价值迭代（Value Iteration）
反复应用 Bellman **最优**算子，直到收敛：
\[
V_{k+1}(s) = \max_a \sum_{s',r} p(s',r|s,a)\big[r + \gamma V_k(s')\big]
\]
收敛后：\(\pi^*(s) = \arg\max_a Q^*(s,a)\)

#### 收缩映射
Bellman 最优算子 \(T^*\) 在 sup 范数下是 **γ-收缩映射**：
\[
\|T^* V - T^* U\|_\infty \le \gamma \|V - U\|_\infty
\]
由 Banach 不动点定理，迭代 \(V_{k+1} = T^* V_k\) **必收敛到唯一不动点** \(V^*\)——即最优价值函数。

## 深度 Q 学习（DQN）

当状态空间巨大或连续时，无法表格存储 Q(s,a)，用神经网络 **Q(s,a; θ)** 近似。

#### 核心算法：DQN
1. 用经验回放（Experience Replay）打破样本相关性
2. 用目标网络 \(Q(s,a;\theta^-)\) 稳定训练目标
3. 最小化 TD 误差：
\[
L(\theta) = \mathbb{E}\Big[\big(r + \gamma \max_{a'} Q(s',a';\theta^-) - Q(s,a;\theta)\big)^2\Big]
\]

#### TD 更新（Bellman 残差）
单步更新形式：
$$
Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \max_{a'} Q(s', a') - Q(s, a) \right]
$$
括号内为 **TD 目标 − 当前估计**，α 为学习率。

**要点**
- **探索 vs 利用**：ε-greedy、UCB 等
- **过估计**：max 操作导致 Q 值偏高 → Double DQN、Dueling DQN
- **连续控制**：DDPG、SAC 等（Actor-Critic 路线）

## 基于模型的强化学习

**Model-Free**（如 DQN）：直接从交互学 Q 或策略，不显式学环境模型。

**Model-Based**：先学环境模型 \(\hat{P}(s'|s,a)\) 或 \(\hat{r}(s,a)\)，再用于规划。

#### 世界模型（World Model）
用神经网络预测「下一状态 / 下一观测 / 奖励」，在**学到的模型里做想象 rollout（imagination）**：
- 减少真实环境交互次数（样本效率更高）
- 支持在 latent space 中规划（如 Dreamer、MuZero 思路）
- 风险：模型误差累积，想象轨迹偏离真实

典型流程：**感知 → 编码为 latent → 预测 dynamics → 在 latent 中优化策略**

## 具身智能基本概念

**具身智能（Embodied AI）**：智能体拥有物理或仿真身体，通过**感知—行动闭环**与环境交互，而非纯文本/纯符号推理。

核心要素：
- **感知**：视觉、深度、力觉、本体感觉等
- **行动**：连续控制（关节、底盘、抓取）
- **环境**：物理规律、碰撞、摩擦、遮挡
- **任务**：导航、操作、多步组合（如「找物—抓取—放置」）

与「离身」AI 的区别：知识来自**交互**，必须处理部分可观测、延迟奖励、安全约束。

#### 具身智能仿真平台

| 平台 | 特点 |
|------|------|
| **Isaac Sim / Isaac Lab** | NVIDIA，GPU 并行、高保真物理，适合 RL 大规模训练 |
| **MuJoCo** | 轻量、接触动力学准确，学术常用 |
| **PyBullet / CoppeliaSim** | 开源、易上手 |
| **Habitat / AI2-THOR** | 室内导航、交互式家务场景 |
| **Gazebo + ROS** | 机器人实机与仿真衔接 |

仿真 → Sim2Real 迁移是工程难点：域随机化、系统辨识、真实数据微调。
